== Description ==

= Track your site with Webtrends =
Easy Webtrends makes using Webtrends with Wordpress easy and straightforward. Once Easy Webtrends has been enabled, you can quickly load your tracking scripts through the Wordpress dashboard (Click Settings -> Easy Webtrends) and start monitoring your visitors and their behaviour through your Webtrends dashboard.

= Compatible with Webtrends v9 & 10 = 
Easy Webtrends is compatible with both Webtrends v9 and v10. Simply generate your tracking code using the <a href="https://tagbuilder.webtrends.com/" title="Webtrends tag builder">Webtrends tag builder</a> and paste the generated script into the Easy Webtrends interface. The plugin will load the correct javascript file and you'll be ready to go in seconds.

= Quickly tag your entire site =
Through the Easy Webtrends options panel, you can quickly tag your entire site. Easy Webtrends will load these tags on each and every page. Its also just as easy to setup custom rules that tag your site based on particular URL strings.

= Load custom tags into your posts and pages =
Easy Webtrends allows you to specify a number of custom tags based on a URL-string match. This will load a user-defined tag (or tags) if the URL of a given page contains a user-defined string.

== Installation ==

The plugin is simple to install:

1. Download 'easy-webtrends.zip'
1. Unzip
1. Upload 'easy-webtrends' directory to your '/wp-content/plugins' directory
1. Go to the plugin management page and enable the plugin
